# Bootcamp API

> Backend API for our bootcamps directory app.

## Usage

Inside config directory , rename  "config.env.example" to "config.env" 
and update the values to your own.


## Install Dependencies
```npm install ```


## Run App
```
 # Dev mode
 npm run Dev

 # Prod
 npm start

```
